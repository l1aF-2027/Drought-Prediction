## How to run it on your own machine

1. Clone this repository
   ```
   git clone https://github.com/l1aF-2027/drought-prediction.git
   cd drought-prediction
   ```

2. For frontend:
   ```
   cd drought-prediction
   npm i
   npm run dev
   ```

3. For backend:
   ```
   cd drought-api
   pip install -r requirements.txt
   cd 'ALL_THE_API_DIR'
   py app.py
   ```
